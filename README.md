# BrightTV-Viewership-Analytics-Case-Study

## Overview
This project analyzes BrightTV’s user and viewership data to generate actionable insights that support the CEO’s objective of growing the subscription base.

The analysis focuses on:
- User behavior and engagement trends  
- Drivers of content consumption  
- Content strategy for low-engagement periods  
- Growth strategies for user acquisition and retention  

---

## Tools Used
- **Databricks SQL** – Data processing and analysis  
- **Power BI** – Dashboard development and visualization  
- **Excel** – Exported aggregated dataset  
- **GitHub** – Version control and documentation
- **Miro** – Project Planning
- **CANVA** – Presentation    

---

## Methodology
1. Data ingestion into Databricks  
2. Data cleaning and preprocessing  
3. Timezone conversion (UTC → South Africa Standard Time)  
4. Exploratory Data Analysis (EDA)  
5. Feature engineering  
6. Aggregation into a final analytical dataset  
7. Export to Excel for Power BI visualization  
8. Dashboard creation and insight generation  

---

## Key Insights
- Peak consumption occurs during evening hours (18:00–22:00)  
- Weekends show significantly higher engagement than weekdays  
- Certain provinces dominate total watch time  
- Gender and age segments display different content preferences  
- Low consumption days correlate with lack of premium or trending content  

---

## Business Recommendations
- Introduce targeted content on low-consumption days  
- Personalize recommendations by demographic segments  
- Offer promotional bundles in low-engagement regions  
- Increase marketing during peak engagement windows
